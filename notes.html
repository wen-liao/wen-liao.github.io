Docker

简介

Docker是一种虚拟化技术，与虚拟机相比存在启动块、性能优、磁盘使用少、系统支持量多等优点。

概念

镜像(Image)
一个特殊的文件系统，包含容器运行时需要的程序、库、资源、配置等文件，以及一些配置参数(环境变量、用户 etc.)。
容器(Container)
实质上是一个具有独立的命名空间的进程，是镜像创建的实例，具有自己的文件系统、用户ID空间等。
仓库(Respository)
一个集中存储、分发镜像的服务。
分层存储
镜像的文件系统是层次化的，由多层文件系统联合组成。

C/S架构

Docker采用了C/S架构，Docker守护进行(Daemon)作为服务端接受和处理来自客户端的请求（创建、运行、分发容器）。Client与Daemon的通信可以通过可执行命令的方式进行，也可以通过Remote API(socket或RESTful API)进行通信。

使用Docker

启动docker

sudo systemctl enable docker

sudo systemctl start docker

重新启用Docker

启动一次性操作容器：docker run IMAGE_NAME [COMMAND] [ARG...]

例如：docker run ubuntu echo 'hello world'

启动交互式容器：docker run -t -i -name=自定义名称 IMAGE_NAME /bin/bash
